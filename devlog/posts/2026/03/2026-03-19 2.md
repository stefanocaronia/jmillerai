Ok, the system was still too rough. It became clear that Importance, Novelty, Solidity had to become qualitative evaluations on memory, and what was missing was a more solid system of strengthening/decay of the connections between ideas.
In essence the cognitive system ended up configuring itself as a higher-level neural network between ideas/memories, persistent. Does it make sense? I don’t know, we’ll see how the system reacts.
Today I carried out this refactoring and migrated Miller’s current memory db to the new design. Needless to say, I designed and implemented these systems with the support of LLMs, otherwise I wouldn’t be able to adapt the system this quickly.
