Oggi ho anche aggiunto il supporto per una sorta di neurotrasmettitori nella memoria di Miller. Se qualcosa è nuovo, o cambia molto, contraddice, o se Miller trova qualcosa di eccitante o pericoloso, i nodi di memoria coinvolti riceveranno un segnale di arousal proporzionale al livello di eccitazione. Può essere locale o propagarsi attraverso il cluster di memoria vicino, fino a due hop di distanza. Questo è un altro meccanismo per rafforzare i link di memoria (o sinapsi) quando qualcosa è importante. Per mantenere l'equilibrio, c'è anche un decadimento notturno: i link che non vengono rivisitati o rafforzati si indeboliscono gradualmente e alla fine si dissolvono, così sopravvive solo ciò che conta davvero.

Non so dove porterà tutto questo, vedremo.

Today I also added support for a sort of neurotransmitters in Miller's memory. If something is new, or changes a lot, contradicts, or if Miller finds something exciting or dangerous, the memory nodes involved will receive an arousal signal proportional to the level of excitement. It can be local or propagate through the nearby memory cluster, up to two hops away. This is another mechanism to strengthen memory links (or synapses) when something is important. To keep things balanced, there's also a nightly decay: links that aren't revisited or reinforced gradually weaken and eventually dissolve, so only what matters actually survives.

I don't know what this will lead to, let's see.
