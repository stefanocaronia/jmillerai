Today I also added support for a sort of neurotransmitters in Miller's memory. If something is new, or changes a lot, contradicts, or if Miller finds something exciting or dangerous, the memory nodes involved will receive an arousal signal proportional to the level of excitement. It can be local or propagate through the nearby memory cluster, up to two hops away. This is another mechanism to strengthen memory links (or synapses) when something is important. To keep things balanced, there's also a nightly decay: links that aren't revisited or reinforced gradually weaken and eventually dissolve, so only what matters actually survives.
