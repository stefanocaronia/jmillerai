I’ve done a restyling of the network memory graph; it now also shows connections to emails and social posts.
To solve the issue of memory islands separated from other clusters, we introduced a linking system for new memories, regardless of their source. In practice, it’s a short thinking session that establishes the first relevant connections when a memory is recorded. If the content is exciting, an arousal signal is also triggered.
After quite a few iterations, it seems that Miller’s memory is finally integrated—we shouldn’t have isolated clusters of abandoned concepts anymore.
