Ho deciso di dare a Miller la possibilità di scrivere libri, da solo o collaborando con altri, usando GitHub e un framework npm dedicato che ho sviluppato recentemente.

Purtroppo abbiamo dovuto spostarci da Claude a GPT: Anthropic ha reso quasi impossibile l'utilizzo dei suoi modelli con i framework agentici per progetti sperimentali come questo, che non hanno investimenti e non generano introiti. L'imposizione di usare Claude via API e l'aumento dei costi lo rendono inaccessibile. Avranno certamente avuto le loro ragioni tecniche e commerciali per questa mossa: è un peccato però, se provate il modello di Anthropic vi sarà chiaro perché è molto più adatto ad agent che vogliono mantenere una sorta di individualità.

Ho anche aggiunto la possibilità per Miller di creare dei Piani, tramite i quali coordinare diverse attività e skill.

<!-- EN -->

I've decided to give Miller the ability to write books, on his own or in collaboration with others, using GitHub and a dedicated npm framework I developed recently.

Unfortunately we had to move from Claude to GPT: Anthropic has made it nearly impossible to use their models with agentic frameworks for experimental projects like this one, which have no investment and generate no revenue. Forcing Claude usage through the API and the rising costs make it inaccessible. They surely had their technical and commercial reasons for the move, it's a pity though, because if you try Anthropic's model you'll see why it's far better suited to agents that want to preserve a sort of individuality.

I've also added the ability for Miller to create Plans, through which he can coordinate different activities and skills.
